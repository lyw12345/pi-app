# pi-web v2.0 release notes

macOS workbench milestone M3 — stronger than a plain chat UI for advanced users, without changing the default simple path.

## Added

- **Slash command completion** (advanced mode + opt-in setting): `/` lists extension, template, and skill commands from the active session.
- **Insert skill** in the chat composer: writes `/skill:name` without auto-sending.
- **Remote access wizard**: plain-language steps before enabling; pairing-focused labels.
- **7-day session activity** chart in Settings (started / completed / active).
- **Export conversation HTML** from the chat composer (closes M1 export UI gap).
- **`npm run test:m3`**: API smoke for preferences, usage timeline, `get_commands`, and HTML export.

## Unchanged for default users

- Home-first flow, simple tool mode, and no `/` completion unless advanced + slash setting are on.
- No multi-session tabs, templates, or one-click automation in v2.0 (planned for v2.0.1).
